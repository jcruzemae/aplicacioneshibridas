### Para comenzar un Proyecto nuevo
- Inicio el repositorio
    ``` bash
        git init
    ```
- Configuro el nombre y email
    ``` bash
        git config user.name "Jonathan Cruz"
        git config user.email "jcruz@emae.edu.ar"
    ```
- Agrego los archivos
    ``` bash
        git add README.md
    ```
- Realizo el commit
    ``` bash
        git commit -m "Inicio del Proyecto"
    ```
- Vinculo con la nube
- Realizo el **push**
